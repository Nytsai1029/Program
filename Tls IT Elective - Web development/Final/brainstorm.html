Brainstorm

Reformatted from Unit 3 · Class 4 to be clearer and easier to read.

Quick Summary

  • Topic: A Beginner’s Guide for Vibe Coding
  • Goal: Teach people how to use AI to support coding
  • Style: Advanced, modern, friendly
  • Palette: White, bright green, dark blue, dark gray

Part 1: Finding Your Passion & Topic

Q1. What are your top 5 hobbies?

  • Coding
  • Piano
  • Reading
  • Gaming
  • Vibing

Q2. For the first two interests above, could you create a beginner’s guide, club/event website, or solve a common problem with a website?

My chosen topic is: A Beginner’s Guide for Vibe Coding

Part 2: Defining Content & Structure

Q3. What is the primary goal of your website?

  • Teach people how to use AI to help coding.
  • Show how to describe requirements clearly.
  • Provide resources and examples for better understanding.

Q4. List at least 3 separate pages your website will have.

  • Home: introduction to vibe coding and its benefits.
  • Tutorials: step-by-step guides on using AI for coding.
  • Resources: links to tools, articles, and examples.
  • Examples: small coding examples with explanations.

Q5. What will the pages look like?

Homepage

A catchy headline and short intro, plus a clear call-to-action button to explore tutorials.

The homepage will use strong animations.

Tutorials Page

A clean list of tutorials with titles, descriptions, and “Start Learning” buttons.

Each tutorial could have a simple sub-page.

Resources Page

Resources grouped by category with links and short descriptions.

Optional: a search bar (future idea).

Examples Page

A gallery-style list of examples with quick explanations and links.

Q6. For your main page (Homepage), what are the main 4 content sections?

  1. Shocking titles: a huge title showing the slogan.
  2. Introduction to vibe coding: overview + benefits.
  3. Success stories: testimonials or short case studies.
  4. Get started now: a call-to-action to tutorials and resources.

Part 3: Interactive & Embedded Elements (Iframe Challenge)

Q7. What interactive tool could a visitor use?

  • Tutorials Page: embed a YouTube video about using AI for coding.
  • Resources Page: embed a resource list of coding agents (idea).

Part 4: Designing the Look & Feel

Q8. What color scheme will you use?

  • Vibe/personality: Advanced, Modern, Friendly
  • Main: White
  • Secondary: Bright Green
  • Background: Dark Blue
  • Other: Dark Gray

Q9. Where will you use CSS animations?

My specific animation idea is: Colors and size changes (like an update effect)

Q10. Final project proposal

My website, Vibe Coding Guide, will be about A Beginner’s Guide for Vibe Coding. Its goal is to teach people how to use AI to help coding. It will have pages for Home, Tutorials, Resources, and Examples. The design will feel advanced, modern, and friendly using a color scheme of white, bright green, dark blue, and dark gray. I will enhance it with embedded elements like a YouTube tutorial video and a resource list, plus CSS animations like color and size changes.

Reflection (Short)

Color + Font

Dark blue keeps the site calm and modern; bright green highlights important actions. A clean sans-serif font matches a “tech guide” style.

Advanced Features

I used a CSS-only dropdown menu for navigation and responsive layout rules so the pages look good on smaller screens.
